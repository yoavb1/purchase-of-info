Assume that you are a nurse in a hospital.

{% elif screen == 2 %}

Instructions

The patient's condition is represented by a number. The higher the number, the more likely an intervention is needed.

Since the number represents the probability that the patient needs an intervention, the threshold for an intervention might change from patient to patient.

You will now need to classify 20 events, each referring to a specific patient.

Decide whether the patient needs an intervention based on the provided number.

{% elif screen == 3 %}

Instructions

  • If an intervention was needed and you identified that correctly: +{{ v_tp }} point. If you missed it: -{{ v_fn }} points.

  • If an intervention was not needed and you identified that correctly: +{{ v_tn }} points. If you missed it: -{{ v_fp }} points.

Instructions

In this block, you will again classify 20 patient events based on a number indicating probability of needing intervention.

You will also receive output from an Automatic Health Monitoring System which provides two types of outputs:

  • "Alarm" (red) = system says intervention needed

  • "No Alarm" (green) = system says no intervention needed

{% elif screen == 5 %}

Instructions

The output of the monitoring system will be displayed on a rectangle, where a red rectangle indicates an "Alarm" output and a green rectangle indicates a "No Alarm" output.

Note that the monitoring system is not perfect, and its identification is not always correct.

Remember, after each decision you make, you will see a message that informs you whether your decision was correct or wrong, and your points will be updated accordingly.

  • If an intervention was needed, and you identified that correctly, you get {{ v_tp }} point, but if you missed it you lose {{ v_fn }} points.

  • If an intervention was not needed, and you identified that correctly, you get {{ v_tn }} points, but if you missed it you lose {{ v_fp }} points.
